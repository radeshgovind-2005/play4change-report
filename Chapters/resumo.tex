U!REKA Play4Change é uma plataforma educacional gamificada e multiplataforma que combina jogos sérios com inteligência artificial para promover a aprendizagem em sustentabilidade e educação digital. A aplicação móvel, disponível em Android e iOS a partir de uma única base de código desenvolvida com Kotlin Multiplatform e Compose Multiplatform, apresenta aos jogadores tarefas diárias geradas automaticamente por um modelo de linguagem de grande dimensão através da biblioteca LangChain4j integrada com a API Mistral. As tarefas são produzidas em processos noturnos em lote e armazenadas numa base de dados PostgreSQL com extensão pgvector para desduplicação semântica, garantindo a diversidade do conteúdo. A autenticação é gerida através de OAuth~2.0 com rotação de tokens JWT. O motor de pontuação atribui recompensas e penalizações para incentivar o envolvimento dos utilizadores. O backend é implementado com Spring Boot e implantado via Docker Compose, suportando configuração flexível por domínio temático e nível de audiência.
