U!REKA Play4Change is a multiplatform gamified educational platform targeting sustainability and digital education, developed as a final year undergraduate project. Built with Kotlin Multiplatform and Compose Multiplatform, it delivers a unified Android and iOS experience from a single codebase, eliminating duplication of platform-specific logic. The platform presents players with AI-generated daily tasks produced by nightly batch jobs using LangChain4j and the Mistral language model API, enabling zero cold-start latency at runtime. Tasks are stored in a PostgreSQL database extended with pgvector for semantic deduplication, ensuring content novelty over time. Player identity is managed through OAuth~2.0 with JWT token rotation, providing secure and stateless authentication. A server-side scoring engine drives engagement through configurable rewards and penalties. The backend, built on Spring Boot and deployed via Docker Compose, supports flexible configuration by subject domain and audience level, making the platform reusable across different educational networks.
